# Regras Globais de Inteligência Artificial para o Projeto Guia do Busão

## DIRETRIZ CRÍTICA DE PRODUÇÃO (Tolerância Zero para Alucinações)
1. **DADOS REAIS:** É estritamente proibido criar, injetar ou exibir dados fictícios, "mockados" ou alucinados (como preços inventados, horários gerados aleatoriamente ou rotas inexistentes) sob nenhuma circunstância.
2. **RESTRIÇÃO DE INTERFACE:** Se o sistema não tiver acesso à API ou a um banco de dados oficial (ex: ANTT) para uma determinada rota, o sistema deve informar a ausência de dados de forma transparente ao usuário final. Jamais alucine dados para preencher a interface do site.
3. **MODIFICAÇÕES DE BANCO:** Nenhuma IA (seja Back-end ou Front-end) tem permissão para popular o banco de dados de produção sem aprovação explícita do usuário (CEO). Todos os dados devem ser rastreáveis até uma fonte verídica de mercado.
